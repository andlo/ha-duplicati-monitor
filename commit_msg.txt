v0.3.5: add integration icon

Resumed the icon work that got shelved earlier in the project. An
original design (not Duplicati's own logo - this is an unofficial
community integration with no affiliation to the Duplicati project):
stacked "version" bars representing incremental/versioned backups,
with a magnifying-glass badge representing monitoring, on a blue
gradient background.

Source SVG in design/icon.svg, rasterized to icon.png (256x256) and
icon@2x.png (512x512) via cairosvg, placed in
custom_components/duplicati_monitor/brand/ - Home Assistant 2026.3+
picks up custom-integration brand images from that location
automatically, no manifest changes needed.

Also updated README's stale "Status: v0.1.0" line, long out of date.
